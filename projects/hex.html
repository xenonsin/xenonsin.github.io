About the project

From December 5-7th, I participated in the Orange County Virtual Reality Meetup.

At this point, Nate purchased both an Oculus Rift and the Leap Motion so we wanted to try and incorporate both. Unfortunately, by using the Leap Motion the player had no free hand to use for movement. So we tried to make a shooting gallery style game where you made hand gestures to conjure and cast spells.

While Nate was able to handle the gesture detection of the Leap motion, Here's what I was able to accomplish:

  • • Enemy AI and pathfinding.
  • • Homing spells.
The game was made using Unity, Leap Motion, and the Oculus Rift, written in C#. Art assets were created by Gemma using Paint Tool Sai.
