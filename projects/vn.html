About the project

From April 3-5, I participated in LA Hacks, which was held in UCLA.

This Visual Novel is a proof of concept built upon a Dialogue System I wrote during the event. The Dialogue System parses and collects data from a spreadsheet. It allows for a non-programmer friendly way for designers to write content for the game. The system is dynamic enough to run entirely from the spreadsheet. Also, the dialogue display is heavily inspired by Pillars of Eternity which was released around the time of the event. The dialogue builds up, and the player is able to scroll up and review what has been said.

This event made me realize that good art is a huge boost in motivation. When the game you're making looks amazing, it really makes you determined to finish.

Here's what I was able to accomplish:

  • • Developed a dynamic dialogue system based on an excel spread sheet.
  • • Copied the dialogue display system found in Pillars of Eternity.

CVS Script

The game was made using Unity written in C#. 2D Art assets were created by Shaina using Paint Tool Sai. Story and Excel Spread Sheet written by Thuy.
