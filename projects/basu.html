About the project

On January 24th, I participated in the Global Game Jam, which was held in USC.

Basu Fighter is a 3D local multiplayer fighting game which is inspired by the old Kirby minigames. The goal of the game is to try and knock each other off the platform.

Before this event, I created a team of people from my club who are completely new to game development. I guided them to follow basic tutorials and helped them out whenever they asked, but never took an active programming role. I took this opportunity to learn how to 3D model using youtube tutorials.

Here's what I was able to accomplish:

  • • Learned how to model, rig, texture, and animate a 3D Model.

global game jam page

The game was made using Unity written in C#. 3D Art assets were created by me using Maya 2015.
