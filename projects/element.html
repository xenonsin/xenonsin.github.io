Project Element
Teammates: Josue Herrera, Nate McDorman
About the project

Project Element is a 2.5D procedural platfomer. The player starts off without any skills and gains skills as he advances through the different levels defeating bosses. Each element gives the player an attacking projectile, and a platfoming skill, they can use to advance through the level.

A large part of the design of both the skills and the bosses is inspired heavily from Megaman.

Here's what I was able to accomplish:

  • • 2D Character Controller
  • • Generic Monster AI
  • • 4 unique boss AI which illustrate the type of element they command
  • • 4 unique platforming skills that are representative of each element
  • • 3D models for both the player, enemy, and the bosses
The game was made using Unity written in C#. 2D Art assets were created by Josue and Nate using Photoshop. 3D Models were created by me using Maya 2015
