Lights Out
Teammates: Chet Manjunath, Daniel Ramirez, Gemma Busoni, Jeffrey Wong, Justin Liwag
About the project

On January 21, 2014 I participated in CodeDay LA. It was my first hackathon/game jam type event so I really didn’t know what to expect. I pitched an idea for a 2D puzzle platformer game with the intention of inviting anyone interested in learning how to make games using unity or learn how to code using c#. Justin came up with the idea of transitioning between worlds, and to achieve this I used unity's tagging system to tag the level assets as either "black" or "white" to disable/enable them whenever the user "turns the light off". We designed the early levels to slowly ease, and introduce the player to the game's mechanics. First showing him the concept of being able to turn off the light, next showing the player that certain objects exist in only one world, then showing that some boxes can be moved around in order to reach different areas.

Our team managed to win top overall so I'd say it was a great first experience for all of us.

The game was made using Unity and written in C#. Transitioning between worlds concept designed by Justin. Main menu screen was created by Daniel and Gemma. All art assets were made by Chet using Photoshop.
