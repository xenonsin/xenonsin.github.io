Boba Finder
Teammates: Fan Yang, Po Lin Chen, Jeffrey Chiu
About the project

There are a lot of people including all of our team members who like to drink the boba milk tea. Therefore we wanted to create an app that was user-friendly and convenient way to find the user’s favorite boba tea. Our group has found an app similar to what we had in mind. However, this app limited its boba search to only stores in Malaysia. What we wanted to create was an app that could be used freely and at any time. So,we decide to develop this Boba Finder.

With just a push of a button, our application would be able to find the user’s boba without having to surf the Internet. Using the phone’s gps system and Yelp’s database, the app will compile a list of stores/restaurants that sells boba. Then by using google map we can show where the store is located to and how to go there from the user’s current location.

Although we can create a database ourselves, this would take far too much time. With a seven weeks deadline, there would be no way that we would be able to store every boba store around the country. Therefore for the sake of time and convenience, we used Yelp’s database to offer immediately and reliable data. Thankfully, Yelp’s API is available for free and for the public to use. That’s how we will make the Boba Finder to work.

google play page

This app was made using Java in Android Studio, while incorporating the Yelp API and Google Maps API.
